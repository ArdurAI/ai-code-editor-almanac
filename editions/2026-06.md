## Code Editors — Terminal Agents Take Over

### The state of the category

The AI coding market is projected at $26–27B by 2030. Over 60% of professional developers use AI coding assistants daily. The defining shift of 2026: **from autocomplete to autonomy**. Terminal-native agents (Claude Code, Codex CLI, Gemini CLI, OpenCode, Cline, Kimi CLI) are the fastest-growing subcategory.

### Findings
- **Multi-agent parallel execution** is the next frontier: Google Antigravity Manager Surface, Claude Code Agent Teams, Kimi CLI Agent Swarm (300 sub-agents).
- **Pricing turbulence**: GitHub Copilot's switch to usage-based credits broke billing predictability. Cursor heavy users report $40–50/mo on a $20 plan.
- **Security risks scaling**: 1 in 5 orgs experienced a serious security incident from AI-generated code. 45% of AI-generated samples had OWASP Top 10 vulnerabilities.
- **EU AI Act compliance gap**: August 2, 2026 deadline; most vendors have not caught up.
